Methodology

Why 5 variations of the same concept isn't enough

Current AI image tools generate variations. We generate perspectives.

🚫

The Problem

Ask any AI tool for "5 LinkedIn images" and you get 5 variations of the same visual concept. Same composition, same color palette, same metaphor. It feels like progress, but it's creative tunnel vision.

🎭

The Innovation

The Synthetic Viewer Panel generates 5 creative director personas — each with a unique philosophy, scene type, composition style, and color mood. Every persona interprets the same post differently, like a real agency pitch.

✨

The Result

5 genuinely different visual directions. A minimalist might use stark negative space while a cinematographer creates an epic landscape — for the exact same post. You choose the direction, not just the variation.

How It Works

FAQ

Common questions

How are the personas generated?+
Dynamically by Claude, based on the specific post content. The skill analyzes your post's theme, tone, and audience, then generates 5 creative director personas — each with a unique scene type, composition style, and color mood. No two runs produce the same set of personas. The diversity rules ensure genuine creative range.
How many API calls does this use?+
7 Gemini API calls per post: 5 for the initial panel (one per persona) and 2 for refining the top picks. The persona generation itself happens inside Claude's reasoning — no additional API calls needed for that step.
Can I customize the persona archetypes?+
Yes. Edit the SKILL.md file to add your own example persona sets, modify the diversity rules, or change the style guidelines. The examples in the skill are calibration references, not hardcoded templates.
What Gemini model does it use?+
gemini-3.1-flash-image-preview by default for the best balance of speed and quality. For high-stakes posts, you can switch to gemini-3-pro-image-preview for maximum quality.
Does this work with Claude Code only?+
Yes, it is a Claude Code skill (.claude/skills/ directory). It requires Claude Code's ability to execute bash commands and use the Read tool for image review. It will not work in the Claude web interface or API directly.
