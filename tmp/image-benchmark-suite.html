Methodology

Grading Model

Gemini 2.5 Pro -- Google's best stable vision model for evaluative reasoning. Chosen for consistent, calibrated scoring across visual dimensions.

Scale Design

0-5 per dimension, based on research from arXiv 2601.03444. Explicit level descriptors (anchors) at each score point prevent score compression and ensure the full range is used.

Chain-of-Thought

The model describes its observations for each dimension before assigning a numeric score. This forces deliberate reasoning and produces auditable justifications.

Anti-Bias Measures

  • "Use the full range 0-5" instruction
  • Anti-self-enhancement prompt
  • Anchored descriptors prevent central tendency
  • Temperature=0, topK=1 for deterministic output

Scoring Formula

Weighted score = sum of (dimension_score / 5 * weight * 100). Grade scale: 90+ = A+, 85+ = A, 80+ = A-, 77+ = B+, 73+ = B, 70+ = B-, 67+ = C+, etc.
