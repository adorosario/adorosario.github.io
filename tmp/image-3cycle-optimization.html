Key Finding: Cycle 2 Is the Sweet Spot

The optimization loop shows a clear pattern: Cycle 2 produces the best results in 4 out of 5 topics. The average jumps from 73.4 to 83.8 (+10.4 points) after incorporating grading feedback. But Cycle 3 regresses in 4 out of 5 topics — the model over-corrects when given too many accumulated fixes.

Recommendation: Use a 2-cycle loop (generate → grade → regenerate with feedback), then pick the best of cycles 1 and 2. This "pick-the-best" strategy averages 87.4 vs the human website average of 78.1 — a statistically significant +9.3 point advantage.

What We Learned

Cycle 2 is the sweet spot

4/5

topics peak at cycle 2. One round of feedback is optimal. The model integrates 3 specific fixes well, but accumulating 6+ fixes causes confusion and regression.

Cycle 3 usually hurts

-13.0

average drop from cycle 2 to 3. The model over-corrects when given too many accumulated instructions. "Fix X, Y, Z, ALSO fix A, B, C" overwhelms the generator.

"Pick the best" strategy wins
87.4

avg when selecting the best cycle per topic, vs 78.1 for human. Generate 2 cycles, grade both, publish the higher scorer. Simple and effective.

AI wins 4 out of 5 matchups

+7.0

average point advantage for AI best-cycle vs human. The only loss (no-login, -3 points) had visual quality issues — an unlucky generation, not a systemic problem.

Typography is the persistent weakness

3.2 / 5

avg typography score across all AI images. Text rendering in image generation models is still imperfect — the main area where humans maintain an edge.

Brand personality comes naturally

4.8 / 5

avg personality score. When given explicit brand rules, the model consistently produces friendly, tech-forward imagery. The brand prompt template works.

Methodology & Technical Details ▾

Image Generation: gemini-2.0-flash-exp-image-generation via Google AI Studio API.

Image Grading: gemini-2.5-pro (best stable vision model) with structured JSON output.

Rubric: 8 dimensions, 0-5 anchored scale (arXiv 2601.03444). Chain-of-thought before scoring. Anti-self-enhancement and "use full range" instructions.

Temperature: 0 with topK=1 for grading (maximum determinism). Default for generation.

Weights: Brand Color 15%, Layout 15%, Visual Quality 15%, Icons 15%, Typography 10%, Personality 10%, CTA/Badge 10%, Cohesion 10%.

3-Cycle Loop: Cycle 1 = brand prompt only. Cycle 2 = brand prompt + top 3 fixes from C1 grading. Cycle 3 = brand prompt + all fixes from C1 + C2.

Human baseline: OG images from 7 customgpt.ai blog posts, graded with the same rubric.

Known biases: Self-enhancement bias (5-7% AI inflation) mitigated with explicit prompt instruction. Score clustering mitigated with anchored descriptions.

Scripts in repo: scripts/grade-image.sh — portable grading script. examples/benchmark-reference/ — 7 human reference images with manifest.
