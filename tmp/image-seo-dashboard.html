Pipeline Architecture (7-Pass)

PassComponentModelInputOutput
1 Vision Classifier gpt-4.1-mini Raw image (base64) type, decorative flag, relevance, description
2 Vision OCR + Describe gpt-5.3-codex Raw image (base64) 2-4 sentence factual description with OCR
3 Entity-First AEO Alt Text gpt-5.3-codex Vision desc + page context [Entity] + [relationship] + [detail] (80-125 chars)
4 LLM-as-Judge Grader gpt-5.3-codex Alt text + vision desc 6-dim rubric score → overall /10
4b Remediation Loop gpt-5.3-codex Previous alt + critique Improved alt (max 3x until ≥ 8/10)
5 ImageObject Schema gpt-5.3-codex Alt text + vision + URL schema.org/ImageObject JSON-LD
6 Figcaption Generator gpt-5.3-codex Alt text + vision + title Visible caption (60-120 chars)
Caption exclusions: Featured/hero (pos 0) • CTA components (blog-cta--container) • CTA-linked images (<a href="register">) • Logos • Icons • Width < 400px
Quality gate: All alt text must score ≥ 8/10 on 6-dimension rubric (entity placement, information gain, semantic accuracy, factual tone, length, citation quality)
